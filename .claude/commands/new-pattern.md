with a descriptive filename like order-followup.md:

# Pattern Name

- [condition 1]
- [condition 2]

## Recommendation
[Specific tool and action to recommend]

## Expected Time Savings
[X hours per week]

## Setup Steps
1. [Step 1]
2. [Step 2]
3. [Step 3]

## Estimated Monthly Cost
[$ amount or free]
