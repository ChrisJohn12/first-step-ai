# Post-Purchase Follow-Up Automation

## Trigger Conditions
- Order status changes to "fulfilled" or "delivered"
- Customer has placed 1 or more orders (applies to all buyers)
- Order value exceeds $25 (filters out low-value noise)
- Store processes 50+ orders per month

## Recommendation
Use **Klaviyo** (or Shopify Email) to automate a post-purchase flow:
- Send a "thank you + order confirmation" email immediately at fulfillment
- Send a "delivery check-in + review request" email 5–7 days after delivery
- Send a "reorder nudge or related products" email 14–30 days post-purchase

## Expected Time Savings
8–12 hours per week (eliminates manual follow-up emails, reduces support tickets, and automates review collection)

## Setup Steps
1. Connect Klaviyo to your Shopify store via the Klaviyo app in the Shopify App Store
2. In Klaviyo, create a new **Flow** triggered by the "Fulfilled Order" event
3. Add three email steps with time delays: immediate, +7 days, +21 days
4. Customize templates with dynamic order data (product names, tracking link, customer name)
5. Enable a review request in step 2 using a tool like **Judge.me** or **Loox** for social proof collection
6. Set flow live and monitor open/click rates in the Klaviyo analytics dashboard

## Estimated Monthly Cost
- Klaviyo: Free up to 500 contacts, then ~$20–$45/month for 1,000–5,000 contacts
- Judge.me (reviews): Free tier available; $15/month for full features
- **Total: $0–$60/month** depending on list size and review tool tier
