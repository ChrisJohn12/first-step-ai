# First Step AI — Claude Code Skills

## Available Commands

### /new-pattern
Creates a new e-commerce automation pattern file
in /src/data/patterns/
Usage: /new-pattern [description of seller pain point]
Example: /new-pattern abandoned cart recovery for 
Shopify seller doing 200 orders per month

### /build-intake-question
Creates a new intake question component and updates
questions.js
Usage: /build-intake-question [question description]

### /new-component
Creates a new React component with Tailwind styling
Usage: /new-component [ComponentName and description]
