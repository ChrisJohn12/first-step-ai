# Product Review Generation for Etsy Sellers

## Trigger Conditions
- Order status changes to "delivered" (estimated 3–7 days after shipment)
- Store processes ~75 orders/month with low review conversion rate
- Seller relies on organic Etsy search visibility and is not ranking well
- Fewer than 30% of buyers leave a review without prompting

## Recommendation
Use **eRank** or **Marmalead** for listing SEO combined with **Etsy's built-in "Message to Buyers"** automation and a follow-up message via **Etsy Conversations** (manual or via a tool like **Sale Samurai**):
- Auto-send a "thank you + review request" message 5–7 days after estimated delivery
- Include a direct link to the order review page to reduce friction
- Personalize with the product name to increase response rate

## Expected Time Savings
3–5 hours per week (eliminates manual follow-up messages, increases review volume passively, and improves Etsy search ranking over time)

## Setup Steps
1. In Etsy Seller Hub, go to **Shop Manager → Settings → Info & Appearance** and enable "Message to Buyers" — write a warm thank-you note that encourages leaving a review
2. Set up a follow-up message template in **Sale Samurai** (or manually via Etsy Conversations) triggered 5–7 days post-delivery, referencing the specific item purchased
3. Keep the ask simple: one clear sentence requesting a review with a direct link — avoid sounding transactional
4. Optimize listings with **eRank** (free tier) to improve search visibility alongside the review increase
5. Monitor your review rate monthly in Etsy's Stats dashboard and adjust message timing or copy if conversion stays below 20%

## Estimated Monthly Cost
- Etsy Message to Buyers: Free (built-in)
- Sale Samurai: ~$9.99/month (optional, adds automation and analytics)
- eRank: Free tier available; $9.99/month for Pro
- **Total: $0–$20/month**
